# 🚀 Quick Start Guide - Interactive Dashboard

## Running the Complete Analysis

### Step 1: Run Full Analysis Pipeline
```bash
python src/main.py
```

This will generate all analysis outputs in the `outputs/` directory:
- ✅ Timeline and participant statistics
- ✅ Graph network metrics
- ✅ Collaboration bursts (7 detected with adaptive parameters!)
- ✅ Project milestones (8 detected)
- ✅ Phase transitions (4 detected)
- ✅ Sentiment analysis (47 events analyzed)
- ✅ Influence rankings (42 participants ranked)
- ✅ Handoff events (detected team changes)

### Step 2: Launch Interactive Dashboard
```bash
streamlit run src/visualization/dashboard.py
```

The dashboard will open in your browser at `http://localhost:8501`

## Dashboard Features

### 📊 Overview Tab
- **Key Metrics**: Total events, participants, emails, meetings, bursts
- **Interactive Timeline**: Scatter plot of all communications
- **Top 10 Participants**: Ranked by activity
- **Event Distribution**: Pie chart of email vs meeting ratio

### 🔥 Bursts Tab
- **Burst Timeline**: Visual highlighting of collaboration bursts
- **Burst Details**: Expandable cards with metrics for each burst
  - Event count, participant count, duration, confidence score
- **With Adaptive Parameters**: Now detects 7 bursts instead of 1!

### 🎯 Milestones Tab
- **Milestone Timeline**: Scatter plot by confidence and date
- **Milestone Types**: Decision points, deliverables, planning phases
- **All Milestones Table**: Sortable list with confidence scores

### 📈 Phases Tab
- **Phase Evolution**: Visual flow of project phases
- **Topic Keywords**: What the team focused on in each phase
- **Transition Details**: 
  - Planning → Design → Scoping → Maintenance
  - Confidence scores for each transition

### 💭 Sentiment Tab
- **Overall Distribution**: Pie chart of positive/neutral/negative
- **Sentiment Statistics**: Average sentiment score
- **Trend Over Time**: Monthly sentiment changes
- **Score Scale**: 0 (negative) to 1 (positive)

### 🕸️ Network Tab
- **Top 10 Influencers**: PageRank-based rankings
- **Role Classification**:
  - 🔴 Active Leaders (high influence + high activity)
  - 🔵 Strategic Leaders (high influence + moderate activity)
  - 🟢 Executors (low influence + high activity)
  - ⚪ Contributors (moderate influence + activity)
- **Interactive Network Graph**: Zoom, drag, explore connections
- **Graph Statistics**: Nodes, edges, density, average degree

## Troubleshooting

### Dashboard won't load?
```bash
# Make sure streamlit is installed
pip install streamlit

# Check if outputs exist
ls outputs/*.csv
```

### Missing data?
```bash
# Run analysis first
python src/main.py

# Should create 10+ CSV files in outputs/
```

### Network visualization not showing?
```bash
# Install pyvis
pip install pyvis
```
